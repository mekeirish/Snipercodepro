# 🎯 Snipercode | L'IDE Ultra-Ciblé

> Propulsé par **Mekeirish** • Partie de *The Code Suite*

Snipercode est un environnement de développement chirurgical et ultra-léger. Conçu pour le *Vibe Coding*, il permet d'ouvrir, d'éditer et de formater des blocs de code ou des fichiers monolithiques entiers de manière instantanée, sans la lourdeur d'un IDE traditionnel.

## 🔮 Interface & Design
*   **Charte Graphique :** `The Code Suite - Strict` (Matte Cyber & Soft UI).
*   **Coloris :** Zone d'édition fusionnée avec le fond noir mat profond (`#0b0b0d`). Coloration syntaxique contrastée mais douce, boutons de contrôle en Violet Électrique.
*   **Layout :** Cockpit de tir minimaliste centré à 100% sur le code.

## 🚀 Fonctionnalités Clés
*   **Édition Immersive :** Mode focus absolu pour manipuler de longs blocs HTML/JS sans distraction.
*   **Commandes Géométriques :** Actions rapides regroupées (Sauvegarder, Recharger, Formater) avec retours mécaniques fluides (`active:scale-[0.98]`).
*   **Indicateurs de Précision :** Micro-badges en Jaune Cyber affichant en temps réel la ligne courante, le statut du fichier et le compteur de caractères.
